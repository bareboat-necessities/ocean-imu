# ocean-imu
Collection of algorithms for IMU data processing in marine environment. Spectral wave analysis, boat heave, wave direction, wave height, attitude (yaw, roll, pitch), tilt compensated compass, GPS fusion, NMEA sensors with IMU, marine AHRS/INS, marine Kalman filters, Arduino/MCU code for marine IMU. Aimed for performance with measurable quality gates on test cases with simulated data and Latex formatted documentation converted into PDF for each release.
